# 📧 AI-Powered Mail Automation using n8n & Gemini LLM

This project extends a basic mail automation workflow by integrating a **Basic LLM Chain with Gemini AI** in n8n. It fetches user data from Google Sheets and generates **personalized email content based on each user's personality**.

## 🔹 Features

* Fetch recipient details from Google Sheets
* Generate AI-based personalized email content
* Unique messages for each user based on their character
* Ready-to-use email drafts for sending

## 💡 Learning Outcome

* Integration of LLMs with automation workflows
* Using AI for dynamic content generation
* Enhancing personalization in communication
